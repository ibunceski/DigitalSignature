# DigitalSignature - Application for the Integrated Systems course project

## Requirements
* .NET 10 SDK

## Setup
1. Clone the repository:
```
git clone https://github.com/ibunceski/DigitalSignature.git
cd DigitalSignature
```
2. Restore dependencies:
```
dotnet restore
```
3. Build the project:
```
dotnet build
```

## Running the project
1. Open the solution file (.slnx) in your IDE.
2. Set the Web.Api project as the Startup Project.
3. Run the project.
